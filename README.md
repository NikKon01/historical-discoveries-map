# Визуализация исторических плаваний российских мореплавателей

Веб-приложение: интерактивная карта (Leaflet) + временная шкала для 20 географических
открытий российских мореплавателей XVII–XX вв. Данные с фотографиями и нейро-озвучкой.

## Возможности
- Маркеры-портреты мореплавателей на карте (фолбэк — инициалы, если фото нет).
- Клик по маркеру → карточка с фото, описанием и ссылкой на Википедию.
- Маршруты с подписями ключевых точек; плавный «полёт» кораблика вдоль пути с озвучкой.
- Озвучка описаний нейроголосом (Silero TTS), синхронизированная с движением кораблика.
- Фильтры по типу экспедиции и веку + ползунок времени.
- Кнопка «Воспроизвести» — анимация появления открытий год за годом.
- Викторина «Кто этот мореплаватель?» — 8 вопросов по портретам со счётом.

## Структура проекта

| Файл / папка | Назначение |
|--------------|------------|
| `index.html` / `styles.css` / `app.js` | Само веб-приложение |
| `data/explorers.seed.json` | Исходные данные: имя, год, координаты, статья Википедии |
| `data/routes.json` | Маршруты (точки с названиями) и категории |
| `data/explorers.json` | Итоговые данные (генерируются скриптами) |
| `data/audio/*.mp3` | Озвучка описаний (генерируется) |
| `scripts/fetch.mjs` | Подтягивает фото и описания из Wikipedia REST API |
| `scripts/merge.mjs` | Вмердживает маршруты и категории в данные |
| `scripts/tts_silero.py` | Генерирует озвучку через Silero TTS |
| `scripts/serve.mjs` | Локальный статический сервер |

## Требования

| ПО | Назначение | Проверка версии |
|----|------------|-----------------|
| **Node.js** ≥ 18 | сборка данных и веб-сервер | `node --version` |
| **Python** ≥ 3.9 | генерация озвучки | `python --version` |
| **ffmpeg** | конвертация аудио в MP3 | `ffmpeg -version` |
| Интернет | только при первой сборке (Wikipedia + модель Silero) | — |

Python-пакеты для озвучки:

```bash
pip install torch omegaconf
```

> `ffmpeg` должен быть доступен в PATH. Windows: установить с https://ffmpeg.org
> или `winget install Gyan.FFmpeg`.

## Быстрый старт

Если данные и аудио уже сгенерированы (папки `data/` и `data/audio/` на месте),
достаточно запустить сервер:

```bash
node scripts/serve.mjs
```

Открыть в браузере: **http://localhost:8080**
(остановить сервер — `Ctrl+C` в терминале).

## Полная сборка с нуля

Выполнять по порядку из корня проекта:

```bash
# 1. Фото и описания из Wikipedia  (нужен интернет)
node scripts/fetch.mjs

# 2. Маршруты и категории
node scripts/merge.mjs

# 3. Озвучка через Silero TTS  (первый запуск качает модель ~38 МБ)
python scripts/tts_silero.py

# 4. Запуск приложения
node scripts/serve.mjs
# открыть http://localhost:8080
```

### Озвучка (Silero TTS) — детали
- Смена голоса: `python scripts/tts_silero.py xenia`
  (доступные: `baya`, `kseniya`, `xenia`, `aidar`, `eugene`).
- Файлы кладутся в `data/audio/<id>.mp3`, путь дописывается в `data/explorers.json`
  (поле `audio`).
- В приложении длительность плавания кораблика берётся из реальной длины аудио,
  чтобы рассказ успевал договориться.
- Если аудиофайла нет — кораблик просто плывёт без звука (браузерный синтез не используется).

> **Примечание.** У хостинга моделей Silero просрочен SSL-сертификат, поэтому скрипт
> отключает проверку сертификата только на время разовой загрузки весов модели
> (файл затем кэшируется локально). См. комментарий в `scripts/tts_silero.py`.
